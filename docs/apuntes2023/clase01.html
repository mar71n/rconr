Vectores

Los vectores solo pueden contener valores de un mismo tipo. Todos numericos, o todos caracteres, etc.

c(1, 4, 7, 3)
## [1] 1 4 7 3
c("a", "23", "abc")
## [1] "a"   "23"  "abc"
c(TRUE, FALSE, FALSE, TRUE)
## [1]  TRUE FALSE FALSE  TRUE
vnumeros <- c(1500, 4750, 7300, 3250, 4701, 3302, 5200)

Podemos extraer el valor almacenado indicando la posicin o ndice:

con [

# el segundo elemento:
vnumeros[2]
## [1] 4750
# el segundo y el quinto elementos:
vnumeros[c(2,5)]
## [1] 4750 4701
# del quinto al septimo elementos:
vnumeros[5:7]
## [1] 4701 3302 5200
vnumeros[TRUE]
## [1] 1500 4750 7300 3250 4701 3302 5200
#[1] 1500 4750 7300 3250 4701 3302 5200
vnumeros[c(FALSE, TRUE, TRUE, FALSE, TRUE, FALSE, TRUE)]
## [1] 4750 7300 4701 5200

Tambien pordemos extraer los valores que cumplan alguna condicin:

La funcin base::which nos responde qu ndices son verdaderos.

# Al vectorizar, R evala la comparacin elemento por elemento
# vector booleano con TRUE donde el elemento es mayor que 4000, y FALSE en caso contrario.
vnumeros > 4000
## [1] FALSE  TRUE  TRUE FALSE  TRUE FALSE  TRUE
# elementos que cumplen la condicin
vnumeros[vnumeros > 4000] 
## [1] 4750 7300 4701 5200
# ndices de los elementos que cumplen la condicin
which(vnumeros > 4000)
## [1] 2 3 5 7
vnumeros[which(vnumeros > 4000)] 
## [1] 4750 7300 4701 5200
# ndice del menor elemento seguido del ndice del segundo valor, y as..
order(vnumeros)
## [1] 1 4 6 5 2 7 3
# elementos ordenados
vnumeros[order(vnumeros)]
## [1] 1500 3250 3302 4701 4750 5200 7300
# elementos ordenados
sort(vnumeros)
## [1] 1500 3250 3302 4701 4750 5200 7300
vnumeros[c(TRUE, FALSE)]
## [1] 1500 7300 4701 5200
#[1] 1500 7300 4701 5200

Matrices

Las matrices, igual que los vectores, solo pueden contener valores de un mismo tipo. Todos numericos, o todos caracteres, etc.

matriz <- matrix(c(18, 8.10, 20, 9, 21, 9.45, 22,   9.90, 23,   10.35), ncol = 2, byrow = TRUE )

matriz
##      [,1]  [,2]
## [1,]   18  8.10
## [2,]   20  9.00
## [3,]   21  9.45
## [4,]   22  9.90
## [5,]   23 10.35
# La columna 1
matriz[,1]
## [1] 18 20 21 22 23
# La fila 3
matriz[3,]
## [1] 21.00  9.45
# El elemeneto guardado en la fila 3, columna 1
matriz[3,1]
## [1] 21
# Las filas desde la 2 a la 4
matriz[2:4,]
##      [,1] [,2]
## [1,]   20 9.00
## [2,]   21 9.45
## [3,]   22 9.90

Puedo ponerle nombre a las dimensiones de una matriz:

# https://www.argentina.gob.ar/redsube/tarifas-de-transporte-publico-amba-2021

matriz_colectivo <- matrix(
  filas <- c(18, 8.10, 20, 9, 21,   9.45, 22,   9.90, 23,   10.35),
  nrow = 5, byrow = TRUE,
  dimnames = list(c("0 a 3 km", "3 a 6 km", "6 a 12 km", "12 a 27 km", "ms de 27 km"),
                  c("Tarifa"    ,"Con tarifa social"))
)

matriz_colectivo[,"Tarifa"] * 1.4
